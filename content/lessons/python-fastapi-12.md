## Задание 1

Как вы знаете, интерпретатор Python последовательно выполняет исходный код вашей программы.  Если говорить более конкретно, он анализирует текст исходного кода, в ходе анализа составляет список всех действий которые нужно в соответсвии с этим текстом сделать.  При выполнении программы он просто последовательно делает все эти действия.


```python
my_list = [1]
my_list.append(2)
print(my_list)

def fn():
    items = [1, 2]
    items.append(3)
    lst = items
    a = lst[0]
    a += 1
    print(items, lst, a)
    return {"key": "Hello world"}

fn()
```

Что выведет этот код на Python?

Напишите, какие именно действия интерпретатор Python последовательно делает при выполнении этого кода.  Опишите каждое действие настолько подробно, насколько можете.
